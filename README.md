Nike Shoes Web App (Next.js)

A modern Nike-inspired e-commerce website built using Next.js. This project focuses on clean UI/UX, responsive design, and smooth custom animations to deliver an engaging shopping experience.

Features :-
  Built with Next.js
  Clean and modern UI inspired by Nike
  Fully responsive (mobile, tablet, desktop)
  Smooth navigation and layout structure
  Custom animations for better user interaction
  Product listing with categories (Men, Women, Kids, Sale)
  Search bar UI
  Trending and featured product sections
